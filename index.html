Hajk SQL Preparator

The purpose of this small app is to make it easier to prepare SQL queries to be used in Hajk. The idea is simple: enter a configuration, as it would be entered into Hajk's configuration, and the app will generate a valid SQL query, ready to be used in you database.

Note that the generated SQL contains a part that looks like this: :query_string. This string can be replaced in your SQL editor of choice (e.g. DBeaver) with the actual search query string.
